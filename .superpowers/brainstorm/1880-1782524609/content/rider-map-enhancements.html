Rider Map — Enhancement Ideas

Route lines are being removed. Which of these additions would make the map more useful for a dispatcher?

A

Pulse Ring on Active Riders

ACCEPTED and PICKED_UP riders get an animated pulsing ring around their dot — immediately visible at a glance which riders are in motion vs idle. Idle riders stay static.

B

Rider Click Popup

Click any rider dot to see a popup: rider name, current status badge, active order ID (linkable), and how long they've been in that state. Closes on map click.

Live Stats Bar on Map

A compact overlay in the map corner showing real-time counts: Idle / Active / Offline, plus queue depth. Updates every second from socket data. No modal needed — pure at-a-glance.

Cluster Markers When Zoomed Out

When multiple riders are near each other at low zoom, collapse them into a numbered badge ("5"). Expand on zoom-in. Mapbox has native GeoJSON clustering — minimal code, big clarity win for dense areas.
