Order Map — Route Trail Effect

As a rider moves, how should the route line behind them behave?

A

Hard Cut (Recommended)

The route line only ever shows the remaining path ahead of the rider. As the rider moves forward, the line behind them disappears entirely. Clean, focused — tells the dispatcher exactly how much road is left.

origin (done) remaining route → customer

Implementation: slice route coordinate array from current step index forward. Step index comes from the simulation tick payload.

B

Faded Trail

The path behind the rider stays visible but fades to low opacity. The path ahead stays full brightness. You can see where the rider came from and how far they've gone.

faded trail remaining → customer

Two separate GeoJSON features: traveled coords (20% opacity) + remaining coords (100%). Same step-index split.

Active Leg Only — same for both options
